# Pygame Pacman AI

A Pacman-inspired game built with Python and Pygame.

## Features

* Grid-based map system
* Dot collection and scoring
* Multiple ghost enemies
* BFS pathfinding AI
* Win and lose conditions
* Multiple difficulty levels (planned)

## Technologies

* Python 3
* Pygame

## Future Improvements

* Level system
* Multiple maps
* Advanced ghost behaviors
* Power-ups
* Main menu
* Sound effects
